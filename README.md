# CSE 476 Final Project by Benjamin Malamuth

## Requirements

Python 3.13.9 was used to test the script.
To install dependencies with conda:
1. `conda create -n final_project python=3.13.9`
2. `conda activate final_project`
3. `pip install -r requirements.txt`

## Usage

Run the following command to generate the results from the test data:
- `python generate_answer_template.py`
